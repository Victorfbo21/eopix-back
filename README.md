## Descrição do Projeto

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Instalar Dependências

```bash
$ npm install
```

## Rodar Aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
